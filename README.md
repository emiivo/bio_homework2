# bio_homework2

This is the second homework for genome assembling, analysing and annotation of three data sets: ERR204044, SRR15131330 and SRR18214264.
Reference genome: CP015498.

Tools used: 
1. FastQC;
2. Trim Galore;
3. MultiQC;
4. SPAdes;
5. ABySS;
6. RagTag;
7. BWA;
8. Samtools;
9. Gepard;
10. BUSCO;
11. GeneMarkS-2;
12. RAST;
13. BLAST.
14. bedtools;
15. Jalview.
